## Projets Phares

### [Bibiji - Plateforme de Petites Annonces](https://github.com/darkous777/Bibiji-TP2)
Marketplace complet avec système de messagerie intégré, gestion d'images et API RESTful.
- **Technologies** : Flask, MySQL, AJAX, JavaScript
- **Fonctionnalités** : Authentification utilisateur, messagerie en temps réel, recherche avec autocomplétion, panneau d'administration
- **Architecture** : Blueprints modulaires, gestion de rôles, transactions sécurisées

### [AppRobot - Contrôle de Robot Raspberry Pi](https://github.com/darkous777/AppRobot)
Interface de contrôle à distance pour robot mobile avec streaming vidéo.
- **Technologies** : C# (WPF), Python, Raspberry Pi, OpenCV, Flask
- **Fonctionnalités** : Contrôle de mouvement, suivi de ligne, détection d'obstacles, streaming vidéo, lecture de musique
- **Intégration** : Communication TCP/IP, gestion multi-utilisateurs, base de données MySQL

### [Planetarium - Visualisation 3D de Constellations](https://github.com/darkous777/Planetarium)
Application de visualisation 3D interactive avec structures de données avancées.
- **Technologies** : C#, WPF, HelixToolkit, Arbres binaires de recherche
- **Fonctionnalités** : Rendu 3D, parsing JSON manuel, opérations récursives, tests unitaires
- **Algorithmes** : Structures de données complexes, traversées d'arbres

### [Rummikub Digital](https://github.com/darkous777/Rummikub)
Jeu de Rummikub numérique avec adversaires IA.
- **Technologies** : C#, WPF, .NET
- **Fonctionnalités** : Logique de jeu complète, IA, authentification, sauvegarde de scores
- **Projet d'équipe** : Documentation extensive, séparation claire des tâches

### [Library Management - Système de Bibliothèque](https://github.com/darkous777/Library-Management)
Système de gestion de bibliothèque avec PL/SQL.
- **Technologies** : PL/SQL, Oracle Database, Triggers, Packages
- **Fonctionnalités** : Codification automatique, sécurité basée sur les rôles, simulation de transactions
- **Qualité** : Scripts de test complets, gestion des amendes

### [AirLibre - Plateforme de Partage d'Activités](https://github.com/darkous777/AirLibre)
Application web Django pour partager et découvrir des activités communautaires.
- **Technologies** : Django, SQLite, Bootstrap
- **Fonctionnalités** : Authentification, gestion de profils, téléchargement d'avatars, gestion d'activités

### [FTP-Server - Serveur FTP Sécurisé](https://github.com/darkous777/FTP-Server)
Configuration de serveur vsftpd avec chiffrement SSL/TLS.
- **Technologies** : Linux (Ubuntu), vsftpd, OpenSSL
- **Fonctionnalités** : Utilisateurs anonymes, locaux et virtuels, certificats SSL, configuration sécurisée
